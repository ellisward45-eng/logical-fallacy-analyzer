## 📘 Overview

The **Logical Fallacy Detector** is an intelligent reasoning system built by **Walter E. Ward**, designed to identify and analyze logical fallacies in text, audio, and video content.  
It combines rule-based reasoning with deep AI understanding using **Hugging Face Transformers** and **OpenAI Whisper**.

The app supports:
- Text, audio, and video uploads (up to 25MB)
- Secure encryption for all user data
- An **admin dashboard** with control panel tools
- AI-powered reasoning for human-like analysis
- Fully international support for multiple languages

---

## ⚙️ Features

| Feature | Description |
|----------|-------------|
